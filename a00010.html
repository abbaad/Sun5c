Define Documentation

#define MAX_KEYS   6

The maximum number of simultaneous keys.

According to the Sun Keyboard spec, the Type5c implements full N-key Rollover (i.e., all key depressions and releases correctly detected in any sequence regardless of how many keys are being depressed). For now, we will keep this at 6 to be compatible witht the Boot protocol (and the existing code), but in the future, this will increase to 10.

#define SUNKBD_KEY   0x7f

The Sun scancode for 'Idle' (meaning all keys are up).

Also the maximum value for make codes, and the mask to get a make code from a break code.

const uint8_t EEMEM scancodes[]

The scancode table.

Notice that this takes up 128 bytes of EEPROM. Also, the macro key is translated to 0xf9 and can be remapped by the host. The linefeed code has neither an associated key nor an HID code.
